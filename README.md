# 🫐 Zé Açaí

Uma landing page desenvolvida durante um curso de **HTML5** e **CSS3**, com o objetivo de praticar a criação de interfaces modernas, responsivas e semânticas.

## 🚀 Demonstração

🔗 **Acesse o projeto:**  
https://hsantanadev.github.io/ze_acai/

## 📖 Sobre o projeto

O **Zé Açaí** é uma landing page criada para apresentar uma **loja fictícia de açaí**, desenvolvida exclusivamente para fins de estudo e prática em desenvolvimento Front-End.

O projeto tem como foco a aplicação dos principais conceitos de HTML5 e CSS3 na construção de uma página moderna e responsiva.

> **⚠️ Aviso:** A loja, os produtos, as informações e os contatos apresentados nesta página são fictícios e foram criados apenas para fins educacionais.

## ✨ Funcionalidades

- Landing page responsiva
- Layout moderno
- Navegação simples e intuitiva
- Estrutura semântica em HTML5
- Design adaptado para diferentes dispositivos

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3

## 💻 Como executar

Clone o repositório:

```bash
git clone https://github.com/hsantanadev/ze_acai.git
```

Acesse a pasta do projeto:

```bash
cd ze_acai
```

Abra o arquivo `index.html` em seu navegador.

Este projeto foi desenvolvido para praticar:

- Estruturação de páginas com HTML5;

- Organização de arquivos;
- Boas práticas de desenvolvimento Front-End.
